# Simple Magento Workspace for Cloud9.io

### Instalation
1. Fork this repo
2. Sync your Git on Cloud9.io
3. Open your Dashboard
4. Click on Repositories
5. Select Clone to Edit
6. Create your Workspace

### Instructions 
1. Start your workspace
2. Click on Run Project
3. Preview
4. Enjoy ;)
